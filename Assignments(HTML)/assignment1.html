Introduction to Web Development

Web development is process of creating websites and web applications that run onthe internet.It invloves designing ,coding, and maintaining web pages using various technologies


Objectives of Web Development
  • Understand basic structure of HTML document
  • Learn Formatting and text alignment
  • Use lists,images,and hyperlinks

Core Components of a Web Page
  1. HTML
    • Defines structure of the page
    • Uses tags like <html>, <head>, <body>
  2. CSS
    • Used for styling and web page
    • Controls layout, colors
  3. JavaScript
    • Adds interactivity
    • Handles events and validation

Text Formating Options

 HTML allows text to be bold,italic,,underlined, and even colored to make the content attractive.

Special Characters

 Some special symbols used in HTML are:
